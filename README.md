# 考勤與請假規則小幫手

這是一個專為公司同仁設計的打卡與請假時間計算工具。系統基於公司的考勤規則，能自動處理彈性上下班、跨越午休的扣假邏輯，以及遠端會議假等情境。

## 功能特色
- **一般上班試算**：輸入實際打卡時間，自動推算最晚下班打卡時間，並具備遲到與提早打卡提示。
- **一般請假計算**：根據輸入的請假起訖時間，自動扣除 12:00-13:00 的午休時數，並精準計算後續的彈性打卡邊界。
- **遠端會議 (不扣薪假)**：支援以半小時為單位的無薪假時數計算，自動規範最晚應進辦公室的打卡時間。

## 使用方式
本工具完全由純 HTML、CSS 與 JavaScript 開發，無需安裝任何框架或套件。
可直接透過瀏覽器開啟 `index.html` 即可使用，或透過 GitHub Pages 進行線上瀏覽。
